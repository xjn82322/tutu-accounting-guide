## 它能帮你做什么

我把平时用得最顺手的几个功能拎出来讲讲，你别看功能列表长，其实真正记起账来，你只会用到其中三五个。

**截图自动记账** 这个功能是我个人心中的MVP。消费完顺手截个付款截图，App会自动识别金额和支付方式，一键就把账记上了。识别速度挺快，准确度也够，比起手动输入省下一大半时间。

**账单批量导入** 微信、支付宝的账单都能一键导入。懒人福音——一周甚至一个月的账，几分钟就补完了。我有时候出差几天没记账，回来靠这个功能"补作业"，体验很丝滑。

**多账户资产管理** 现金、储蓄卡、信用卡、网上支付、投资理财、债务借出，全都能塞进一个App里管。一眼看清自己有多少钱、欠多少钱，净资产数字实时更新。比起东一笔西一笔的散装记账，这种统一视图对财务状况的把握要清晰太多。

**分类预算 + 还款日历** 给每个支出类别设预算，按年/月/周切换查看，进度条实时预警防超支。还款日历则把信用卡和各种债务到期日统一摆出来，提前通知，再也不用担心逾期影响征信。

**多账本多币种** 旅游账本、装修账本、副业账本可以分开建。百余种货币自动更新汇率，出国消费回国一拉报表，清清楚楚。

**iCloud 多端同步** 数据走iCloud，iPhone、iPad、Apple Watch之间自动同步，不需要额外注册账号。也支持导出CSV，万一哪天想搬家也不至于被锁死。
